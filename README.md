# nixos-config

Personal NixOS system configuration, managed as a flake. Runs Hyprland + [Noctalia](https://github.com/noctalia-dev/noctalia-shell) on a single machine (hostname: `nixos`).

## Structure

```
.
├── flake.nix                  # Inputs: nixpkgs (unstable), home-manager, noctalia, qylock, astronvim-template
├── configuration.nix           # Top-level system config, imports modules/
├── hardware-configuration.nix  # Machine-specific, generated by nixos-generate-config
├── home.nix                    # Top-level Home Manager config, imports home/
├── modules/                    # System-level (NixOS) config, one concern per file
│   ├── boot.nix
│   ├── audio.nix                # PipeWire
│   ├── desktop.nix               # Hyprland, xdg-portal, SDDM
│   ├── hardware.nix              # GPU, bluetooth
│   ├── locale.nix
│   ├── network.nix               # NetworkManager, firewall
│   ├── nix.nix                   # nix.conf settings, garbage collection
│   ├── nix-ld.nix                # dynamic linker shim for unpatched binaries
│   ├── qylock.nix                # SDDM theme + lockscreen
│   ├── services.nix
│   ├── shell.nix                 # enables fish as a valid login shell
│   └── users.nix
└── home/                       # User-level (Home Manager) config
    ├── fish.nix                  # shell abbreviations, fastfetch on launch
    ├── ghostty.nix                # terminal
    ├── hyprland/                  # Hyprland (Lua config format) + Noctalia binds/autostart
    │   ├── core.nix                 # base hl.config(), binds, animations
    │   ├── noctalia.nix              # noctalia autostart, panel/media/lock binds, layer rules
    │   └── default.nix               # imports the above
    ├── neovim.nix
    ├── packages.nix               # user-level packages
    ├── programs.nix                # git config
    └── theming.nix                 # cursor, icon theme, GTK font
```

## Usage

Rebuild the system after making changes:

```bash
sudo nixos-rebuild switch --flake /etc/nixos#nixos
```

Update flake inputs and rebuild:

```bash
cd /etc/nixos && sudo nix flake update && sudo nixos-rebuild switch --flake /etc/nixos#nixos
```

(Both are also available as fish abbreviations: `rebuild` and `update`.)

## Notes

- Uses `nixpkgs-unstable` throughout.
- Hyprland config is written in the newer Lua format (`hl.config()` API), not classic `hyprlang`.
- Theming (colors) is handled dynamically by Noctalia via matugen — not hardcoded here.
- This repo is for reference/backup. Some values (hostname, timezone, cursor theme) are specific to my setup and would need adjusting for another machine.
